# ConnectNow (Talksy)

A login-free random video chat platform using WebRTC and Socket.IO.

## Getting Started

1.  Current directory: `Talksy`
2.  Install dependencies: `npm install`
3.  Start the server: `npm start`
4.  Open `http://localhost:3000` in multiple browser tabs to test.

## Tech Stack
- Frontend: HTML5, CSS3, JavaScript (WebRTC)
- Backend: Node.js, Express, Socket.IO
